### 本地安装流程：

- git clone https://github.com/DustinBrett/daedalOS.git
如果不能克隆，请使用
``` 
git config --global url."https://gh-proxy.com/".insteadOf https://
git clone https://github.com/DustinBrett/daedalOS.git
- cd daedalOS
- npm install -g yarn
- yarn install
- yarn build
- yarn serve
```
